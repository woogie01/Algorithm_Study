1. 원형 연결 리스트 내의 임의의 노드 하나가 주어졌을 때, 해당 List 길이를 구하는 법.
- 동일한 노드가 나올 때까지 계속 다음 노드로 이동.
- 공간 복잡도 : O(1)
- 시간 복잡도 : O(N)

2. 중간에 만나는 두 연결 리스트의 시작점들이 주어졌을 때 만나는 지점 구하는 법.
- 두 리스트에 대하여 각각의 길이를 먼저 구하고,
- 다시 시작점으로 돌아와 더 긴 쪽을 둘의 차이만큼 앞으로 먼저 이동시킨 후,
- 두 시작점이 만날 때까지 시작점에서 동시에 한칸씩 전진.
- 공간 복잡도 : O(1)
- 시간 복잡도 : O(A + B)

3. 주어진 연결 리스트 안에 사이클이 있는지 판단하는 법.
- Floyd's Cycle Finding Algorithm
- 공간 복잡도 : O(1)
- 시간 복잡도 : O(N)
